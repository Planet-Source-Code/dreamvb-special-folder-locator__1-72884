+    Information      +

Microsoft Windows has lots of special folders that are used for storing application
settings and files,Some special folders that you may already know about are  
Internet files,  Pictures, Music, Videos and many more.

With this small program you can displays a list of all special folders 
that are on your system. This makes it much easier to find something that your 
looking for. You can just simply double click the item folder and 
it will jump to the folders location you can also copy the folder path and
create a shortcut to the folder on your desktop.

+---------------------+
+ System Requirements +
+---------------------+

This program has been tested on Windows 2000, Windows XP, Windows Vista, Windows 7
